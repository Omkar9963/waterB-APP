How the model works +
A daily soil-water bucket is solved per pixel. Reference ET comes from ERA5-Land with FAO-56, scaled to crop ET using NDVI (Sentinel-2, gap-filled with MODIS). Rainfall is CHIRPS; runoff uses the slope-adjusted SCS Curve Number. Each day soil moisture = previous + infiltration minus ET, held between wilting point and field capacity, with the excess becoming deep percolation.
Data sources +
ERA5-Land: weather for ET
CHIRPS: daily rainfall
Sentinel-2 / MODIS: NDVI
SoilGrids / OpenLandMap: soil & water-holding
SRTM: elevation & slope
Reading the output +
Layers show the period-mean of each component; readouts are the area-wide daily mean. For big areas the sampling is coarse, so basin/state values are indicative. Export any component as a GeoTIFF.
